*****巴別塔端程式啟動環境設置******
必須先開啟 Windows PowerShell
*路徑
	開始 -> Windows PowerShell -> Windows PowerShell

*執行動作
1.複製以下指令，貼至 Windows PowerShell
cd C:\Users\User\Desktop\8.Tower\VPLab-Light-master\vlc-arduino
按下Enter

2.複製以下指令，貼至 Windows PowerShell
python .\pybackground.py
按下Enter

即可執行

******程式重置************
整館重置需要先重置大腦端，再重置巴別塔端
執行動作如下

1.大腦端
大腦上的按鈕由左至右依序按壓，此處即完成重置。

2.巴別塔端
櫃子上面開關往大腦端扳，此處即完成重置。

******全館關閉*********
大腦端及巴別塔端皆相同

1.關閉背景影片(Black.mp4)
2.切到 Windows PowerShell，按下 ctrl + C ，關閉視窗
